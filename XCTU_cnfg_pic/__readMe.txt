The following pictures are screenshots of the XCTU application that was used to configure the XBee modules for IOT Course Project. 

Here is the breakdown of the naming scheme:

'c1.png': coordinator network AT parameters
'c2.png: coordinator pin configuration AT parameters

'd2-1.png': Xbee2 network AT parameters
'd2-2.png': Xbee2 pin configuration AT parameters

'd3-1.png': Xbee3 network AT parameters
'd3-2.png': Xbee3 pin configuration AT parameters

'd4-1.png': Xbee4 network AT parameters
'd4-2.png': Xbee4 pin configuration AT parameters

'd5-1.png': Xbee5 network AT parameters
'd5-2.png': Xbee5 pin configuration AT parameters
